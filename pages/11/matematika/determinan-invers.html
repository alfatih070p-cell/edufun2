Det 2×2 dan 3×3
Nilai skalar dari matriks persegi
Determinan 2×2
det(A) = ad − bc
A = [[a,b],[c,d]]  |  diagonal utama dikurangi diagonal sekunder
Determinan 3×3 (Sarrus)
det = (aei+bfg+cdh) − (ceg+afh+bdi)
3 diagonal kanan (+) dikurangi 3 diagonal kiri (−)
📌
Sifat Determinan
  • !
    det(A)=0 → matriks singular, tidak punya invers
  • ✓
    det(AB) = det(A)·det(B)
  • ✓
    det(Aᵀ) = det(A)
  • ✓
    det(kA) = kⁿ·det(A), n = ordo
🔄
Invers Matriks 2×2
Syarat: det ≠ 0
Rumus Invers 2×2
A⁻¹ = (1/det) · [[d,−b],[−c,a]]
Tukar a↔d, ganti tanda b dan c, bagi dengan det(A)
Contoh
A=[[2,1],[3,4]], det=8−3=5
A⁻¹ = (1/5)[[4,−1],[−3,2]]
📐
Aplikasi: SPLDV dengan Matriks
Ax = b → x = A⁻¹b
1
Tulis sistem sebagai matriks A dan vektor b
2
Hitung det(A) — jika 0, tidak ada solusi tunggal
3
Hitung A⁻¹, lalu x = A⁻¹b
✏️
Soal Latihan
1
Hitung det dari A=[[3,−1],[2,5]] dan B=[[1,2,0],[3,−1,2],[2,1,−1]].
2
Tentukan A⁻¹ jika A=[[4,3],[3,2]].
3
Selesaikan: 2x+y=5, 3x−2y=4 menggunakan metode invers matriks.
